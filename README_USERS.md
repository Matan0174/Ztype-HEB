# ZType IL - ניהול משתמשים

This update adds user registration, login, and high score tracking with a backend server.

## Installation / התקנה

1.  Open a terminal in the project folder.
2.  Install dependencies:
    ```bash
    npm install
    ```

## Running the Game / הפעלת המשחק

1.  Start the server:

    ```bash
    node server.js
    ```

    Or for development (auto-reload):

    ```bash
    npm run dev
    ```

2.  Open your browser at: `http://localhost:3000`

## Features / פיצ'רים חדשים

- **Registration**: Create a new account.
- **Login**: Log in to an existing account.
- **Persistent High Scores**: Scores are saved to the server.
- **User Interface**: New login/register screens and user status in the main menu.
